The script must follow the exact logic from RESEARCH.md Section 5:

**Section 1 — Dependency checks:**
- Verify `python3` is available (required for patch scripts)
- Verify `jq` is available (required for config.json update)
- Exit with clear error if either is missing

**Section 2 — GSD location detection:**
- Check `$HOME/.claude/get-shit-done` first (global install)
- Check `./.claude/get-shit-done` second (local install)
- Exit with error if neither found: "ERROR: GSD not found. Install GSD first."

**Section 3 — Project root detection:**
- Verify `.planning/` directory exists in CWD
- Exit with error if not found: "ERROR: No .planning/ directory found. Run install.sh from your GSD project root."

**Section 4 — Extension directory setup:**
- `EXT_DIR` = the directory containing install.sh (use `$(cd "$(dirname "$0")" && pwd)`)
- `PROJECT_DIR` = current working directory (`$(pwd)`)
- `EXT_INSTALL_DIR` = `$GSD_DIR/get-shit-done-review-team`
- Create directories: `$EXT_INSTALL_DIR/agents`, `$EXT_INSTALL_DIR/workflows`, `$EXT_INSTALL_DIR/templates`

**Section 5 — Copy extension files:**
Copy only files that exist (use `[ -f ... ]` guards for optional future-phase files):
- Copy agents/*.md if any exist: `[ -f "$EXT_DIR/agents/"*.md ] && cp "$EXT_DIR/agents/"*.md "$EXT_INSTALL_DIR/agents/" 2>/dev/null || true`
- Copy workflows/*.md if any exist: same pattern
- Copy templates/*.md: `cp "$EXT_DIR/templates/"*.md "$EXT_INSTALL_DIR/templates/"`

**Section 6 — Apply patches:**
```bash
echo ""
echo "Applying patches..."

EXECUTE_PLAN="$GSD_DIR/get-shit-done/workflows/execute-plan.md"
SETTINGS="$GSD_DIR/get-shit-done/workflows/settings.md"

if [ ! -f "$EXECUTE_PLAN" ]; then
  echo "  ERROR: $EXECUTE_PLAN not found"
  exit 1
fi

if [ ! -f "$SETTINGS" ]; then
  echo "  ERROR: $SETTINGS not found"
  exit 1
fi

python3 "$EXT_DIR/scripts/patch-execute-plan.py" "$EXECUTE_PLAN"
python3 "$EXT_DIR/scripts/patch-settings.py" "$SETTINGS"
```

**Section 7 — config.json update:**
```bash
CONFIG="$PROJECT_DIR/.planning/config.json"
if [ -f "$CONFIG" ]; then
  tmp=$(mktemp)
  jq 'if .workflow.review_team == null then .workflow.review_team = false else . end' "$CONFIG" > "$tmp" && mv "$tmp" "$CONFIG"
  echo "  [OK] config.json: workflow.review_team ensured"
else
  echo "  [SKIP] No .planning/config.json found (run /gsd:new-project first)"
fi
```

**Section 8 — TEAM.md copy with existence guard (Pitfall 4 from RESEARCH.md):**
```bash
if [ -f "$PROJECT_DIR/.planning/TEAM.md" ]; then
  echo "  [SKIP] .planning/TEAM.md already exists — not overwriting"
else
  cp "$EXT_DIR/templates/TEAM.md" "$PROJECT_DIR/.planning/TEAM.md"
  echo "  [OK] .planning/TEAM.md created from starter template"
fi
```
